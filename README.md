# SnakeRemake
Remake of the original Nokia Snake game
and a two-player game where one snake will move using WASD and another will move using arrow keys.
If Snake A bites snake B then snake B will die, and vice versa.
---features implemeted---
Screen Wrapping
2 Types of Foods
- Mass Gainer
- Mass Burner
3 PowerUps
- Shield → Snake will have a shield and the snake will not die when the shield is active.
- Score Boost → Snake will gain 2x Score Points for each mass-gainer food.
- Speed Up → Snake should increase the speed after collecting this power-up.
  With cooldowns
Co-Op Gamemode
basic Ui and SFX
